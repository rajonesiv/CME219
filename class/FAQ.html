How do I connect to the server/Thomas?

  1. Using your Mac toolbar at the top of your screen, select Go.
  2. Select Connect to Server...
  3. Enter the following into the server address field: smb://thomas.butler.edu
  4. Select Connect
  5. If your computer asks for your username and password, please input that information.
  6. Select your user name folder.
  7. If you are able to access your folder, you have successfully gained access to your webspace on the server.

How do I set up the server connection in Visual Studio Code?

  1. While in Visual Studio Code, select Shift+Command+P and type in the following: SFTP:Config. Hit Enter
  2. This will open up a JSON file. Change the following fields:
    • "host": "thomas.butler.edu"
    • "username": your username
    • "password": your password
    • "remotePath": "/home/username/public_html/CME219"
  3. Hit Save.

How do I publish my webpage to the server/Thomas?

  1. Select the file that you want to upload.
  2. Right-click and choose SFTP: Upload.
  3. A message at the bottom should say Upload Complete

What link do I turn in for assignments?

You will need to turn in the link to your page on the server. NOT your BUFiles location. For instance, if you publish a page called code.html to your class folder, on the server (Thomas), your url will look like this: http://blue.butler.edu/~username/class/code.html


When I view my page/image in a browser, it says Forbidden. How do I correct this?

  1. While in Visual Studio Code, click on the terminal tab at the bottom of your application.
  2. Enter the following text at the prompt: ssh thomas.butler.edu. If you are using your own laptop, type the following text: ssh username@thomas.bitler.edu.
  3. You will then be asked for your password. Even though it appears as if you are not typing your password, you are; the application is password-protecting your password for security reasons.
  4. You should get the following prompt: thomas%
  5. Enter the following text: fix_account
  6. Refresh your browser.
